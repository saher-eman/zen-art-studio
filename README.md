# 🎨 Zen Art Studio — Relax & Create

A beautiful browser-based painting game built with pure HTML, CSS & JavaScript.
No downloads. No installations. Just open and paint!

## 🌟 Features

- 🖌️ Multiple brushes — Round, Soft, Flat, Ink, Glitter & more
- 🌈 Full color palette with custom color picker
- 🎨 6 beautiful animated themes — Sunset, Ocean, Forest, Night, Cherry, Aurora
- 🌟 Stickers & stamps to decorate your art
- 🎵 Ambient music — Rain, Ocean, Forest, Piano, Zen, Fire
- ↩️ Undo / Redo support
- 💾 Save your artwork as PNG
- 📱 Touch screen friendly
